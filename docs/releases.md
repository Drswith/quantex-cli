# Releases

Quantex releases are prepared by release-please Release PRs.

The repository now keeps a source-controlled [CHANGELOG.md](../CHANGELOG.md). A Release PR updates that changelog together with `package.json`, `.release-please-manifest.json`, and `src/generated/build-meta.ts` before publication.

Published release pages and attached binaries live in [GitHub Releases](https://github.com/Drswith/quantex-cli/releases).
